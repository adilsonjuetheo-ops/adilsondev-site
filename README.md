# Adilson Dev

Site profissional estatico para publicar em `adilsondev.com.br`.

## Arquivos principais

- `index.html`: estrutura e conteudo do site.
- `styles.css`: identidade visual e responsividade.
- `script.js`: menu mobile, ano automatico e animacoes de entrada.
- `assets/`: imagens usadas no portfolio.
- `robots.txt` e `sitemap.xml`: apoio basico para SEO.

## Como visualizar

Abra o arquivo `index.html` no navegador.

## Como publicar

Envie todos os arquivos desta pasta para uma hospedagem estatica ou painel de arquivos do provedor.
No Registro.br, aponte o dominio `adilsondev.com.br` para os DNS ou registros informados pela hospedagem escolhida.
